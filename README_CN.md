# Finite 笔记应用（中文说明）

## 项目简介

**Finite** 是一款现代化、跨平台的笔记应用，基于 Tauri + Next.js + React + BlockNote 富文本编辑器开发，支持多端同步、结构化属性、标签、封面自定义、收藏与最近更新等功能。适合个人知识管理、项目文档、任务记录等多种场景。

---

## 技术架构

- **前端框架**：Next.js 15（React 19）
- **富文本编辑器**：BlockNote（支持多列、代码块、Markdown/HTML导入导出）
- **桌面端**：Tauri（跨平台，极致性能）
- **数据库**：SQLite（通过 tauri-plugin-sql 管理）
- **状态管理**：Zustand
- **UI 组件**：Tailwind CSS + Radix UI + Lucide 图标

---

## 主要功能

1. **多端笔记管理**：支持无限分层的笔记树结构，便于知识归类。
2. **富文本编辑**：支持多列布局、代码块、Markdown/HTML 导入导出。
3. **属性自定义**：每条笔记可添加任意属性（文本、数字、日期、单选/多选、链接等），支持自定义选项。
4. **标签与封面**：支持标签管理、渐变色/图片封面自定义、随机封面生成。
5. **收藏与最近**：一键收藏常用笔记，快速访问最近更新。
6. **搜索与命令面板**：支持快捷键（⌘K/Ctrl+K）快速搜索与切换笔记。
7. **多主题与宽屏**：支持暗色/亮色主题切换、宽屏/居中布局自定义。
8. **数据导入导出**：支持 Markdown/HTML 文件导入，内容可导出为 Markdown/HTML。
9. **属性面板**：可视化管理笔记属性，支持增删改查。
10. **快捷操作**：支持快捷键、右键菜单、批量操作等。

---

## 特色亮点

- **极致性能**：Tauri 桌面端，资源占用极低，启动秒开。
- **结构化知识管理**：属性系统高度自定义，支持多种数据类型。
- **美观渐变封面**：内置多种渐变色风格，支持随机生成，提升视觉体验。
- **多列富文本**：BlockNote 多列扩展，适合表格、对比、项目管理等场景。
- **本地数据库**：数据本地存储，隐私安全。
- **可扩展性强**：代码结构清晰，便于二次开发。

## 安装与运行

### 环境要求
- Node.js >= 18
- Rust & Cargo（Tauri 桌面端需要）

### 安装依赖
```bash
pnpm install
# 或 yarn install / npm install
```

### 启动开发环境
```bash
pnpm tauri dev
# 启动 Next.js + Tauri
```

### 打包桌面应用
```bash
pnpm tauri build
```

## 目录结构说明

- `src/components/`：前端 UI 组件（主界面、对话框、侧边栏等）
- `src/lib/`：数据库、类型、工具函数
- `src/hooks/`：Zustand 状态管理 hooks
- `src-tauri/`：Tauri 桌面端相关配置与 Rust 代码
- `public/`：静态资源

## 常见问题与支持

- **数据存储在哪里？**
  - 所有数据本地存储于 SQLite 数据库（`notes.db`），路径见 tauri 配置。
- **如何导入/导出笔记？**
  - 支持 Markdown/HTML 文件导入，支持导出为 Markdown/HTML。
- **如何自定义属性/标签？**
  - 在属性面板可自由添加、编辑、删除属性，支持多选/单选/日期等类型。
- **如何反馈 Bug 或建议？**
  - 请在 GitHub Issue 区提交。
