# Finite

Notion like markdown note-taking app base on [Tauri](https://v2.tauri.app/) and [Next.js]((https://nextjs.org/).

![finite](docs/finite.png)


## Getting Started

To develop and run the frontend in a Tauri window:

```shell
pnpm tauri dev
```

### Building for release


```shell
pnpm tauri build
```
